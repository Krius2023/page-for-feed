Project Overview

This project enables you to feed your pets remotely using IoT (Internet of Things) technology. By leveraging a Raspberry Pi Pico W microcontroller and a Servo SG90 motor, you can automate the process of dispensing pet food from a remote location.

Tools Used

  • Raspberry Pi Pico W
  • Servo SG90
  • USB Power Supply

Setup for Food Feeding

  1. Connect the Servo SG90 to the Raspberry Pi Pico W.
  2. Power up the Raspberry Pi Pico W using a USB cable.

Instructions for Usage

  1. Power on the Raspberry Pi Pico W.
  2. Connect your device to the Wi-Fi network named: NAME.
  3. Open a web browser and navigate to the IP address: 192.168.4.1.
  4. Tap the Feed button on the web interface to remotely dispense pet food.

This project combines IoT technology with pet care, providing a convenient and innovative way to ensure your pets are fed even when you're away from home.
