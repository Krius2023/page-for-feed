IoT based Food Feeding for Pet Animals

This project allows you to feed your lovely pets remotely using IoT technology.

Tools Used:

  • Raspberry Pi Pico W
  • Servo SG90

Setup for Food Feeding:

Just Connect a power suppply with a USB cable.

Instructions:

  1. Power up the Raspberry Pi Pico W.
  2. Connect to the Wi-Fi network named: NAME.
  3. Open the following IP address in your web browser: 192.168.4.1.
  4. Tap the "Feed" button to feed your pets.
